## 字母
1、利用异或操作`^`进行英文字符大小写互转
```java
('d' ^ (1 << 5)) = 'D'
('D' ^ (1 << 5)) = 'd'
```

> 为什么会这样呢？
感兴趣可以参照 ASCII 码：大小写英文字符之间相差 32

由于 32 对应的是 ' '，所以还可以下成如下的形式：
```java
('d' ^ ' ') = 'D'
('D' ^ ' ') = 'd'
```

2、利用或操作`|`将英文字符转换为小写
```java
('a' | ' ') = 'a'
('A' | ' ') = 'a
```
